自动驾驶小车项目说明
该项目是一个基于物联网的小车视觉导航系统。它通过摄像头采集图像，使用计算机视觉技术实现车道跟踪、斑马线检测和红绿灯识别，从而实现小车的自动导航功能。

功能模块
1. 车道跟踪模块
文件名：lane_detection.py
实现功能：识别图像中的车道线，并计算车辆相对于车道中心的偏移量。
使用方法：在图像回调函数中调用该模块的方法即可。
2. 斑马线检测模块
文件名：zebra_crossing_detection.py
实现功能：检测图像中是否存在斑马线。
使用方法：在图像回调函数中调用该模块的方法即可。
3. 红绿灯识别模块
文件名：traffic_light_detection.py
实现功能：识别图像中的红绿灯，判断是否需要停车。
使用方法：在图像回调函数中调用该模块的方法即可。
4. 主文件
文件名：main.py
实现功能：初始化 ROS 节点，订阅图像话题，调用各功能模块完成图像处理，发布控制指令。
使用方法：直接运行该文件即可启动小车的自动导航功能。
使用方法
确保安装了所需的 Python 库和 ROS 环境。
将所有功能模块文件放置在同一目录下。
修改 main.py 文件中的图像订阅话题名称，确保与你的摄像头节点发布的话题一致。
在终端中执行 python main.py 命令，启动小车导航系统。
注意事项
请确保你的摄像头节点正常工作，并发布了正确的图像话题。
在使用过程中，可能需要根据实际情况调整各功能模块的参数，以获得更好的效果。
